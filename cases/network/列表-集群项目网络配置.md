## tc-P1-功能测试：集群-项目网络配置列表数据显示正确

### 前置条件 (pc)
- 集群下存在已绑定 IPPool 的项目

### 测试步骤 & 预期结果
1. 进入【集群】-【项目】，选择某项目进入详情 → 显示项目详情页
2. 切换 Tab 至【网络配置】 → 显示该项目绑定的 IPPool 列表
3. 查看列表字段（名称、状态、类型、网段、已用IP地址） → 各字段显示正确
4. 按名称模糊搜索 → 列表实时筛选匹配结果
5. 按状态筛选 → 列表过滤正确
6. 按类型筛选 → 列表过滤正确
7. 组合查询（名称 + 状态 + 类型） → 结果正确交集
8. 项目未绑定任何 IPPool → 显示空状态提示：当前项目未绑定任何 IPPool，请前往容器组 IP 池页面进行绑定，以便为 Pod 分配 IP 地址
9. 数据超过一页 → 分页控件正常，翻页正确
