## tc-P1-功能测试：容器组列表-IP地址字段显示 IPv4/IPv6/双栈

### 前置条件 (pc)
- 存在已分配 IPv4/IPv6/双栈 地址的容器组

### 测试步骤 & 预期结果
1. 进入【集群】-【工作负载】-【容器组】（或企业空间侧） → 显示容器组列表
2. 查看列表字段（名称、状态、项目、节点、容器组IP地址、更新时间） → 各字段显示正确
3. 查看容器组IP地址列 → 根据 Pod 分配情况正确显示 IPv4/IPv6/双栈 地址格式
4. IPv4 单栈容器组 → 仅显示 IPv4 地址
5. IPv6 单栈容器组 → 仅显示 IPv6 地址
6. 双栈容器组 → 同时显示 IPv4 和 IPv6 地址
