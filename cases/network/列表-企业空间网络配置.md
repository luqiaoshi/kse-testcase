## tc-P1-功能测试：企业空间-项目网络配置列表数据显示正确

### 前置条件 (pc)
- 企业空间下存在项目，且项目已绑定 IPPool

### 测试步骤 & 预期结果
1. 进入【企业空间】-【服务与网络】-【项目网络配置】 → 显示网络配置页
2. 默认自动选中该企业空间下第一个项目（如 host-pro） → 项目下拉框默认值正确
3. 查看列表字段（IP池名称、状态、类型、网络、已用IP地址、IP池块大小） → 各字段显示正确
4. 切换项目下拉框，选择其他项目 → 列表刷新，显示对应项目的 IPPool 绑定数据
5. 按名称模糊搜索 → 列表实时筛选匹配结果
6. 按状态筛选 → 列表过滤正确
7. 按类型筛选 → 列表过滤正确
8. 组合查询（名称 + 状态 + 类型） → 结果正确交集
9. 切换项目后无绑定的 IPPool → 显示空状态提示
10. 切换项目后查询条件重置为默认 → 查询条件不保留（或保留按需求确认）

## tc-P1-功能测试：企业空间-项目网络配置详情数据正确

### 前置条件 (pc)
- 企业空间下项目已绑定 IPPool

### 测试步骤 & 预期结果
1. 进入【企业空间】-【服务与网络】-【项目网络配置】 → 显示列表
2. 点击 IPPool 名称进入详情 → 进入详情页
3. 查看基础信息（IP池名称、状态、类型、网络、已用IP地址、IP池块大小、NAT outgoing、Overlay模式、掩码、起始IP地址、结束IP地址、创建时间、创建者） → 各字段数据正确
4. 切换至【容器组】Tab → 仅显示当前所选项目中使用该 IPPool 的容器组
5. 查看容器组字段（容器组名称、节点、容器组IP地址、CPU监控、内存监控） → 各字段显示正确，IP 地址格式根据类型区分 IPv4/IPv6/双栈
