> 本用例基于 `cases/byd/工作台测试拆解` 中的 Region 管理测试规划生成，覆盖工作台 Region 选择器、平台概览、Region 列表、注册/编辑/移除 Region、Region 监控、权限控制、稳定性、兼容性及国际化等场景。

## 模块一：Region 选择器

### tc-P0-功能测试：工作台 Region 选择弹窗正确展示

#### 前置条件 (pc)
- 已使用 platform-admin 角色登录平台。
- 平台下已存在至少一个 Region。

#### 测试步骤
1. 点击顶部工作台「平台管理」入口
2. 点击「选择 Region」或切换 Region 入口
3. 查看弹窗内容
4. 查看弹窗操作区

#### 预期结果
- 展开平台管理下拉。
- 弹出选择 Region 弹窗。
- 弹窗展示所有 Region，包含名称、描述、状态、成员集群数、同步时间。
- 支持搜索框和刷新按钮，无错位或截断。

### tc-P0-功能测试：选择弹窗搜索 Region

#### 前置条件 (pc)
- 平台下存在多个 Region。

#### 测试步骤
1. 打开选择 Region 弹窗
2. 在搜索框输入目标 Region 名称关键字
3. 输入无匹配的关键字

#### 预期结果
- 弹窗正常展示所有 Region。
- 列表按名称/描述过滤，仅展示匹配项。
- 列表展示空结果提示。

### tc-P0-功能测试：选择 Region 后进入对应 Region

#### 前置条件 (pc)
- 平台下存在状态正常的 Region。

#### 测试步骤

2. 选择某个正常状态的 Region
3. 查看顶部导航

#### 预期结果

## 模块四：注册 Region

### tc-P0-功能测试：注册 Region 基本信息表单字段与校验

#### 前置条件 (pc)
- 已使用 platform-admin 角色登录平台。

#### 测试步骤
1. 进入 Region 列表页，点击右上角「注册」
2. 清空名称字段，输入其他合法信息
3. 输入包含大写字母或特殊字符的名称
4. 输入以连字符开头或结尾的名称
5. 输入超过 63 个字符的名称
6. 输入合法名称，别名和描述分别超过 63/256 个字符

#### 预期结果
- 弹出注册 Region 分步向导，默认进入「基本信息」步骤。
- 名称输入框提示必填，「下一步」按钮不可用。
- 实时或提交时提示格式错误。
- 提示名称格式不合法。
- 提示超出最大长度。
- 别名/描述提示超出最大长度。

### tc-P0-功能测试：注册 Region 名称唯一性校验

#### 前置条件 (pc)
- 平台下已存在 Region `cn-east-prod`。

#### 测试步骤
1. 进入注册 Region 向导的基本信息步骤
2. 输入已存在的名称 `cn-east-prod`，其余字段合法
3. 修改名称为未使用过的合法名称

#### 预期结果
- 表单正常展示。
- 提交或下一步时提示名称已存在。
- 校验通过，可进行下一步。

### tc-P0-功能测试：注册 Region 连接配置必填项校验
