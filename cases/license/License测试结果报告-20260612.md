## Bug 列表

### Bug 1: 概览 API 返回 404
- **用例关联**：tc-P1-功能测试：订阅许可概览字段显示正确
- **接口**：`GET /kapis/license.kubesphere.io/v1alpha1/licenses/overview`
- **实际结果**：
  ```json
  HTTP 404
  {"message": "Secret \"io.kubesphere.license.status.v3.overview\" not found"}
  ```
- **预期结果**：返回概览数据（授权组织、状态、集群ID、授权规模等）
- **严重程度**：中
- **分析**：后端未生成 `io.kubesphere.license.status.v3.overview` Secret，但 UI 能正常显示概览（UI 直接读取前端数据汇总），前端和 API 存在不一致。

### Bug 2: 非管理员可调用 License API
- **用例关联**：tc-P3-接口测试：非管理员调用 License 导入 API → 扩展：非管理员调用任何 License API
- **测试用户**：cluster-viewer / edge-viewer
- **实际结果**：HTTP 200（非管理员同样能获取 License 列表数据）
- **预期结果**：HTTP 403（无权限）
- **严重程度**：高
- **分析**：License 数据涉及集群授权规模、用量等敏感信息，不应允许非管理员查看。
